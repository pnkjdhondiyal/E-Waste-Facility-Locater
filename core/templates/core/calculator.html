📚 How to Find Your Device Specifications

            
            

                

                    
💻 Windows Users

                    
    
                        
  1. Right-click on This PC or My Computer
    2. 
                        
  2. Click Properties
    3. 
                        
  3. You'll see: Processor, RAM, Windows version
    4. 
                        
  4. For GPU: Open Device Manager → Display adapters
    5. 
                    

                

                
                

                    
🍎 Mac Users

                    
    
                        
  1. Click Apple Menu () in top-left
    2. 
                        
  2. Select About This Mac
    3. 
                        
  3. You'll see: Processor, Memory (RAM), Storage
    4. 
                        
  4. Click System Report for more details
    5.
